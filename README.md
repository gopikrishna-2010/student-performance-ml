Student Performance Prediction - Machine Learning Pipeline

Overview
This project builds a machine learning pipeline to predict student academic performance based on demographic, behavioral, and academic features.

 Key Steps
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis (Pearson, Spearman correlation, ANOVA)
- Feature engineering
- Model training using Linear Regression and Random Forest
- Model evaluation using MAE, MSE, and R² score

Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

Objective
The goal of this project is to demonstrate the ability to build an end-to-end machine learning workflow including data analysis, model building, and evaluation.
